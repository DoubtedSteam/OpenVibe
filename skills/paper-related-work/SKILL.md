# 相关工作修改（强约束）

## 工作流
1. 划分 **2-4 个主题组**（基座模型/检索采样/压缩加速/稀疏动态等）。
2. 每组固定结构：总结 -> 2-4 个代表作 -> **limitation 句（必须）** -> 桥接到本文角度。
3. 末尾增加一段 “closest work vs ours（最接近工作 vs 我们）” 的差异总结。

## 输出要求
- 输出 **分组大纲**（每组标题 + 该写哪些内容）。
- 为每组输出一条可直接粘贴的 **limitation+bridge 句**（限制 + 桥接）。

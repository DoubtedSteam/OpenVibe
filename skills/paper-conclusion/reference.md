# Conclusion（结论）强模板

## 强模板（3 句）
说明：以下是常用英文结论句型模板，**模板本身可用英文**（便于直接粘贴到论文里），但逻辑与禁忌必须按本文执行。

1) In this paper, we propose … (method name) for … (task).
2) Our key idea is … (mechanism).
3) Extensive experiments across … demonstrate … (effectiveness) while … (efficiency).

## 禁止项
- 引入新模块/新设定/新实验
- 引入新数字（除非复述已出现的主结论数字；默认不写数字）

## Checklist（pass/fail）
- [ ] 有方法名
- [ ] 有机制短语
- [ ] 有双结论（有效性+效率）
- [ ] 无新内容

## 验收字段（给 Skill 做结构化生成/验收）
- has_method_name
- has_mechanism_phrase
- has_dual_conclusion
- introduces_new_content（必须为 false）
